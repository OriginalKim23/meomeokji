# 머먹지

**머뭇거리는 시간을 줄여주는 메뉴 추천 서비스**

머먹지는 “오늘 뭐 먹지?” 앞에서 머뭇거리는 시간을 줄여주기 위한 메뉴 추천 웹서비스입니다.
사용자가 간단한 질문에 답하면, 상황에 어울리는 메뉴 하나와 비슷한 선택지를 추천합니다.

## 현재 버전

현재 버전은 **v0.1 기능 테스트 버전**입니다.

완성본은 아니며, 메뉴 추천 흐름과 기본 사용성을 확인하기 위한 테스트 단계입니다.
현재는 기능 테스트, 모바일 화면 개선, 추천 로직 개선을 중심으로 개발하고 있습니다.

## 배포 링크

아래 링크에서 테스트할 수 있습니다.

배포 링크: 여기에 Vercel 링크 붙여넣기

## 주요 기능

* 6개 질문 기반 메뉴 추천
* 식사, 디저트, 야식, 아무거나 선택 가능
* 혼자 / 둘 이상 / 상관없어 조건 반영
* 배달, 외식, 집밥, 편의점·간단히 선택 가능
* 예산 조건 반영
* 오늘 당기는 느낌 반영
* 현재 컨디션 반영
* 메인 추천 메뉴 1개 제공
* 비슷한 대안 메뉴 3개 제공
* 추천 이유 표시
* 메뉴 변형 추천 표시
* 다시 골라줘 기능
* 별로야 기능
* 공유하기 버튼
* 모바일 화면 최적화 진행 중

## 실행 방법

프로젝트를 내려받은 뒤 아래 명령어를 실행합니다.

```bash
npm install
npm run dev
```

실행 후 브라우저에서 아래 주소로 접속합니다.

```text
http://localhost:5173/
```

## 프로젝트 목표

머먹지는 단순히 메뉴를 랜덤으로 뽑아주는 서비스가 아니라,
사용자의 상황과 기분을 바탕으로 메뉴 결정을 도와주는 작은 결정 도우미를 목표로 합니다.

처음부터 완성형 서비스를 만드는 것보다,
주변 사람들이 링크로 접속해서 가볍게 테스트해보고 “은근 괜찮다”고 느낄 수 있는 서비스를 만드는 것이 현재 목표입니다.

## 마스코트

머먹쥐는 메뉴 추천을 도와주는 작은 다람쥐 캐릭터입니다.
숟가락을 들고 사용자의 메뉴 고민을 함께 덜어주는 역할을 합니다.
