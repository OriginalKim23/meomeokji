# 머먹지

**머뭇거리는 시간을 줄여주는 메뉴 추천 서비스**

머먹지는 “오늘 뭐 먹지?” 앞에서 머뭇거리는 시간을 줄여주기 위한 메뉴 추천 웹서비스입니다.
사용자가 간단한 질문에 답하면, 상황에 어울리는 메뉴 하나와 비슷한 선택지들을 추천합니다.

## 현재 버전

현재 버전은 **v0.1 기능 테스트 버전**입니다.

아직 최종 완성본은 아니며, 아래 기능들이 실제로 잘 작동하는지 확인하기 위한 초기 테스트 단계입니다.

* 질문 흐름이 자연스러운지
* 사용자의 선택에 따라 추천 메뉴가 달라지는지
* 추천 결과가 납득되는지
* 모바일에서 버튼을 누르기 편한지
* “다시 골라줘”, “별로야” 기능이 자연스럽게 작동하는지

앞으로 테스트 피드백을 바탕으로 디자인, 캐릭터, 메뉴 데이터, 추천 로직을 계속 개선할 예정입니다.

## 배포 링크

스마트폰과 PC에서 아래 링크로 접속해 테스트할 수 있습니다.

배포 링크: 추후 추가 예정

## 주요 기능

* 6개 질문 기반 메뉴 추천
* 사용자의 상황, 예산, 기분, 먹는 방식 반영
* 메인 추천 메뉴 1개 제공
* 비슷한 대안 메뉴 3개 제공
* “다시 골라줘” 기능
* “별로야” 기능
* 머먹쥐 캐릭터를 활용한 귀여운 사용 흐름

## 질문 흐름

1. 오늘은 뭐가 필요하지?
2. 누구랑 먹지?
3. 어떻게 먹을까?
4. 얼마 정도로 생각 중이지?
5. 오늘은 어떤 느낌이지?
6. 지금 상태는 좀 어때?

## 실행 방법

프로젝트를 내려받은 뒤 아래 명령어를 실행합니다.

```bash
npm install
npm run dev
```

실행 후 브라우저에서 아래 주소로 접속합니다.

```text
http://localhost:5173/
```

## 프로젝트 목표

처음부터 완성형 서비스를 만드는 것보다,
주변 사람들이 링크로 접속해서 가볍게 테스트해볼 수 있는 메뉴 추천 서비스를 목표로 합니다.

현재는 추천 흐름과 기본 로직을 확인하는 단계이며,
이후에는 메뉴 데이터 확장, 캐릭터 이미지 적용, 추천 이유 개선, 모바일 디자인 개선을 진행할 예정입니다.

## 마스코트

머먹쥐는 메뉴 추천을 도와주는 작은 다람쥐 캐릭터입니다.
숟가락을 들고 사용자의 메뉴 고민을 함께 덜어주는 역할을 합니다.

## 앞으로 추가하고 싶은 기능

* 메뉴 데이터 확장
* 추천 점수 로직 개선
* 픽셀 스타일 머먹쥐 캐릭터 이미지 적용
* 냉장고 뒤지는 로딩 화면 개선
* 결과 화면 디자인 개선
* 공유용 결과 카드
* 테스트 피드백 반영
